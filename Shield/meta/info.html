UC Arduino Shield Template

PCB template for use in UC projects that involve creating a Shield for the Arduino Uno. Provides a board outline and connectors suitable for use with the Arduino Uno R3, and shows a UC Arduino outline as a user drawing.

Derived from an example Arduino Shield project originally developed by Richard Lane, which appears to have been constructed using the KiCad 4 Arduino Uno template by Jonathan Iapicco.

Converted back from a project into a template by Allan McInnes 2021-2025:

  • Removed unused circuit elements from project schematic and PCB
  • Updated components and DRC exclusions to match current KiCad version (5, 6, 7, 8)
  • Revised net labels to make them easier to use (moved PWM signifiers into text)
  • Renamed mounting hole footprints to clarify use
  • Added meta-information
  • Moved Arduino Uno R3 keepout areas to User.ArduinoUnoR3 layer
  • Added UC Arduino keepout areas and board outline on User.Drawings layer
